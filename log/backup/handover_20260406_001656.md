## 2026-03-08 ~ 2026-03-09 (略)

## 2026-03-10 (視覺化革命與診斷基石)
### 1. 本次對話達成
- **樂器定義帽子積木**：完全對齊 #processing 邏輯，實作「定義音色 -> 引用演奏」的模型。
- **SVG ADSR 視覺化**：開發了自定義 Blockly 欄位 `FieldADSR`，能在積木上繪製曲線並同步顯示演奏進度的「黃色跳動光點」。
- **即時示波器 (Oscilloscope)**：在主介面實作了磁滯觸發示波器，能靜止、穩定地觀察 Rust 輸出的原始波形。
- **音質診斷與優化**：
    - 解決了 Windows 多聲道環境下的緩衝區空洞雜音。
    - 實作了聲部物理隔離（Gate 物理斷路），消除了單音演奏時的並聯滲漏（拍音）。
    - 實作了頻率平滑 (`follow` 節點)，大幅改善了音符切換時的 Pop 聲。

### 2. 技術細節
- **磁滯觸發**：Rust 端的數據抓取採用了 `-0.05 ~ +0.05` 的磁滯區間，確保前端繪圖絕對靜止。
- **SVG vs Canvas**：棄用 Canvas 嵌入 SVG，改用 Blockly 原生 `createSvgElement`，解決了工作區縮放與 Minimap 顯示問題。

### 3. 下一步行動
- **核心穩定化**：目前正弦波在極快速連續演奏時仍有細微雜訊，需研究相位硬同步（Hard Sync）或在 Rust 端實作更細緻的 Crossfade。
- **功能擴充**：實作取樣器 (Sampler) 與 FFT 頻譜分析儀。

## 2026-04-04 (UI 佈局優化與 MDI 狀態同步)

### 1. 右側面板佈局修正
- **雙欄分析儀**：將示波器與頻譜由上下分割改為左右並列分割 (`flex-direction: row`)。
- **收合一致化**：修復了示波器與日誌收合後高度不一的問題，透過 `:not(.collapsed)` 隔離 `min-height` 並強化 `.collapsed` 強制高度鎖定。
- **捲軸功能修復**：日誌面板改回固定 `height: 180px` (非收合狀態)，解決了 `height: auto` 導致捲軸消失的問題。
- **微型 UI 優化**：縮小了收合按鈕 (`mini-btn`) 至 `20px` 並將圖示統一縮放為 `14px`。

### 2. MDI 分頁同步與執行中斷機制
- **切換自動停止 (Euthanasia)**：實作分頁切換時自動執行 `WaveCodeAPI.reset()`，確保舊分頁的非同步演奏腳本立即終止，避免多個分頁同時發聲的衝突。
- **鍵盤演奏同步**：
    - 切換分頁時，自動掃描新工作區的樂器定義 (`scanInstruments`) 並同步至 `WaveCodeAPI`。
    - 解決了 PC 鍵盤在切換分頁後仍使用舊樂器或在新檔下無聲的問題。
- **開新檔案體驗優化**：將 `createDefaultBlocks` 改為同步執行，確保新分頁在建立瞬間即可完成 Patch 初始化，讓使用者在按 Run 之前就能用鍵盤試聽預設音色。

### 3. 下一步行動
- **音訊核心穩定性**：持續監控並消除正弦波快速演奏時的極細微雜訊。
- **教學內容充實**：準備更多配合雙欄分析儀的教學範例專案。
