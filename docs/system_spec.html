WaveCode IDE 系統規格書 (Polyphonic & Visualization)

1. 專案架構概覽

WaveCode 是一個基於 Tauri v2 框架的獨立音訊編程 IDE。它採用 100% 原生 Rust 實作的音訊引擎，具備低延遲、複音演奏與實時視覺化能力。

前端 (腦)
Vite + Blockly + SVG Visualization
後端 (神經)
Tauri v2 + Emitter (Event Bus)
音訊引擎 (身體)
fundsp + cpal (8-Voice Polyphony)

2. 核心技術機制

  • 腦與身體分離：異步邏輯（JS）處理樂譜排程，同步渲染（Rust）處理物理波形。
  • 物理隔離複音架構：每個聲部具備獨立的 gate 物理開關與參數空間，徹底杜絕閒置聲部的雜訊滲漏。
  • 硬同步 (Hard Sync)：每次觸發音符時，相位重置為 0，確保波形連續性，消除切換爆音。
  • 實時示波器 (Oscilloscope)：採用磁滯升緣觸發 (Hysteresis Edge Trigger)，確保前端波形顯示靜止穩定。

3. 視覺化組件

  • ADSR Field：內嵌於積木的 SVG 繪圖組件，具備動態黃色光點隨音符進度移動。
  • Waveform Canvas：主介面頂部的 60FPS 示波器，顯示 Rust 引擎傳回的原始採樣點。

4. 模組化設計

  • ui/src/modules/visualizer.js：封裝 ADSR 欄位定義、動畫管理員與示波器繪製邏輯。
  • src-tauri/src/engine.rs：實作具備 8 聲部輪替與物理斷路機制的音訊引擎。
