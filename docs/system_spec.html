1. 核心技術棧 (Tech Stack)

函式庫 語言 主要用途
fundsp Rust DSP 運算核心，負責所有音訊節點 (OSC, Filter, Net) 的處理。
cpal Rust 跨平台音訊 I/O，負責與 Windows WASAPI 驅動直接對接。
hound Rust WAV 檔案解碼器，支援 16/24/32-bit 取樣讀取。
puremp3 Rust 純 Rust 實作的 MP3 解碼器，用於鋼琴音色庫的讀取。
rayon Rust 數據並行庫，用於程式啟動時利用多核心 CPU 進行背景音色並行載入。
Tauri v2 Rust/JS 原生視窗容器與橋接層，處理檔案系統與前端指令通訊。
Blockly JS 視覺化編程介面。

2. 高品質取樣器技術 (Sampler v2)

  • 線性插值變速 (Linear Interpolation)：
    • 自定義 VariableSpeedPlayer 節點，解決了變速播放時的數位鋸齒雜訊。
    • 支援動態比率 (Ratio) 調整，實現鋼琴、小提琴的多取樣音高映射。
  • 預先載入與背景並行：
    • 程式啟動後自動建立背景執行緒掃描音色庫。
    • 利用 rayon 實現多執行緒並行解碼，數百個音檔可在 3 秒內完成加載。
  • 音量標準化 (Normalization)：
    • 載入期自動執行 Peak Normalization。
    • 針對打擊樂 (Kick) 提供額外增益補償 (+1.5x)，平衡與 MP3 取樣間的響度差異。

3. 系統穩定性機制

  • 重啟音訊引擎 (Rocket Launch)：解決 Windows 筆電睡眠喚醒後常見的 WASAPI 時鐘偏移雜訊。
  • 全域硬停止 (Hard Stop)：Stop 指令不僅關閉 Gate，還會立即將所有活躍聲部替換為靜音單元，切斷長 Release 尾音。
  • 'static 生命週期管理：使用 Box::leak 管理全域音色 Arc 引用，根除音訊執行緒非法記憶體存取的閃退 (Panic) 風險。

4. 介面與 UX 規格

  • 多取樣映射：選擇 Piano (Multi) 時，引擎自動根據頻率匹配最接近的錄音檔。
  • 系統設定選單：整合語系切換 (i18n) 與重啟功能，支援滑鼠懸停展開子選單。
  • 更新系統：具備逆時針旋轉與彈跳色彩漸變動畫，實時反應版本狀態。
