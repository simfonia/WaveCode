WaveCode IDE 系統規格書 (Pure Rust Refactor)

1. 專案架構概覽

WaveCode 是一個基於 Tauri v2 框架的獨立音訊編程 IDE。它捨棄了外部依賴（如 Pure Data），採用 100% 原生 Rust 實作的音訊引擎。

前端 (腦)
Vite + Blockly + Version Control API
後端 (神經)
Rust (Tauri v2 Commands)
音訊引擎 (身體)
fundsp + cpal

2. 核心技術機制：腦與身體分離

WaveCode 採用「異步邏輯與同步渲染分離」的設計：

[Blockly UI] |-- (編譯分析) --> [Compiler.js] -> (啟動引擎音量) |-- (執行腳本) --> [WaveCodeAPI] -> (傳送頻率/演奏事件) | v [Tauri Commands] -> [AudioEngine (Rust)] -> [fundsp Graph] -> [Speaker]
  • 無鎖通訊：主執行緒透過 Shared 變數與音訊執行緒通訊，確保音訊輸出無跳轉 (Glitch-free)。
  • 版本管理 (Execution ID)：前端 _execId 確保多次點擊 Run 時，舊的演奏任務會自動中斷，杜絕音符重疊問題。
  • NaN 防護盾：透過攔截 setAttribute 確保 Blockly 座標異常不會導致 UI 崩潰。

3. 模組化設計

  • ui/src/modules/api.js：封裝所有給積木呼叫的非同步指令與版本檢查。
  • ui/src/modules/ui_utils.js：負責 NaN 防護、Minimap 初始化與 Resize 管理。
  • src-tauri/src/engine.rs：定義 fundsp DSP 鏈條與無鎖控制變數。

4. 音訊鏈擴充計畫 (Roadmap)

  • 複音混音器 (Voice Mixer)：支援同時播放多個音色。
  • ADSR 包絡線：控制聲音的啟閉平滑度，避免生硬爆音。
  • 效果器鏈條：實作 Delay, Reverb 等 fundsp 原生效果。
