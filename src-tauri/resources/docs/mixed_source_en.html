Mixed Source

The Mixed Source block in WaveCode combines a stable oscillator with textured noise. Powered by the Rust-based fundsp engine, it provides a rich foundation for creating "organic" or "analog" synth sounds.

Engine Performance: Like all WaveCode sound sources, the Mixed Source supports 8-voice polyphony, allowing for complex, noisy chord textures without dropouts.

Parameters

  • Wave: Provides the stable base pitch (Sine or Saw).
  • Noise: Provides texture (White or Pink noise).
  • Mix Level (Noise %): The percentage of noise in the final mix. High values are great for percussion or wind-like effects.

Advanced: Creating Lo-Fi Tones

Mixed Source provides the foundation. To achieve a true Lo-Fi effect, add these effects in sequence:

  1. Mixed Source (Wave + 10% Noise)
  2. Pitch-Mod: Use "Jitter" mode to add instability.
  3. Auto-Filter: Add dynamic breathing to the frequency spectrum.
