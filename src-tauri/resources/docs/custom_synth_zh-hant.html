進階音色設計：加法合成原理

除了使用基礎波形，WaveCode 的 fundsp 引擎 允許我們透過「傅立葉級數 (Fourier Series)」的概念，利用多個正弦波疊加來打造極其複雜的音色。這稱為 加法合成 (Additive Synthesis)。

1. 諧波與泛音列 (Harmonics)

自然界中的聲音（如鋼琴或小提琴）並非純粹的頻率。它們是由一個「基頻」以及許多「倍頻」組合而成的。

  • 1 倍頻 (Fundamental)：決定您聽到的音高。
  • 2, 3, 4... 倍頻 (Overtones)：決定聲音的「厚度」與「音質」。

2. 非整數倍率與金屬質感

當疊加的頻率不是基頻的整數倍時，會產生「不諧和」的聽感：

  • 整數倍率：產生和諧、溫暖的管風琴或銅管音色。
  • 非整數倍率 (如 1.414 或 2.718)：產生具有金屬感、鐘聲或工業噪音般的質感。

物理與數學的交界

在音訊程式碼中，我們實際上是在計算空氣壓力的隨時間變化函數：

y(t) = A₁·sin(2πf₁t) + A₂·sin(2πf₂t) + ...

在 WaveCode 的 Rust 引擎中，這類運算被高度優化，即使疊加數十個振盪器，依然能保持極低的延遲與純淨的音質。

性能提示： Rust 引擎能輕鬆處理大量的並行運算。在 WaveCode 中，每個樂器定義都獨立運行在自己的數位訊號鏈中，這確保了複雜音色不會干擾到整體的系統穩定度。
