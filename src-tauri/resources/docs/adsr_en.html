ADSR Envelope

In WaveCode, the ADSR envelope is a core component of the Rust-based fundsp engine. It defines the volume (or filter) curve for each of the 8 polyphonic voices from the moment a note starts until it ends.

1. Parameter Definitions

ParameterNameDescription
A (Attack)AttackThe time it takes for the sound to reach its peak level. Small values create percussive hits; large values create slow fades.
D (Decay)DecayThe time it takes for the sound to drop from the peak to the Sustain level.
S (Sustain)SustainThe constant volume level held while a key is pressed (0.0 - 1.0).
R (Release)ReleaseThe time it takes for the sound to fade to silence after the key is released.
Engine Logic: Because WaveCode uses fundsp nodes, these parameters are updated at the sample level, ensuring ultra-smooth transitions without digital clicking.

2. Common Sound Presets

  • Piano: Very fast attack, moderate sustain, moderate release.
    A: 0.01, D: 0.3, S: 0.4, R: 1.0
  • Strings: Slow attack (fade-in), high sustain, slow release (long tail).
    A: 0.5, D: 1.0, S: 0.8, R: 1.5
  • Drums: Instant attack, fast decay, zero sustain.
    A: 0.005, D: 0.1, S: 0.0, R: 0.1

3. Visual Feedback

In WaveCode, you can observe the real-time status of the envelope via the visual chart on the block. A yellow light point will dance according to the playback progress of each voice.
