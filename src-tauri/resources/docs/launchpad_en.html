MIDI and Launchpad Control

The WaveCode Engine features a robust Rust-based MIDI implementation. It supports standard MIDI communication and is optimized for the Novation Launchpad series, enabling real-time interaction between hardware and software.

1. Setup

Init MIDI

  • Input: Set the index of the MIDI input device (Launchpad is usually 0).
  • Output: Set the index of the MIDI output device. This must be set correctly to control Launchpad LEDs.
Tip: Check the device list printed in the Rust Terminal after running to find the correct indices for your hardware.

2. Handling Input

When MIDI Note ON / OFF received

Triggered when a key on the hardware is pressed or released. These events are processed on a dedicated high-priority thread in the WaveCode Engine to ensure minimal latency.

3. Controlling LEDs (Output)

Send MIDI Note messages to the Launchpad to control its LED colors. WaveCode uses Rust's asynchronous I/O to manage these messages efficiently.
