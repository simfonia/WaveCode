Melody List Writing Guide

The "Play Melody List" block in WaveCode uses a simplified text format. It supports single notes, chord names, and powerful Tie features.

1. Note Names & Octaves

A note consists of a Note Name (A-G), Accidentals (#/b), and an Octave (0-9):

  • Format: e.g., C4 (Middle C), G#3 (G sharp), Bb5 (B flat).
  • Case Insensitive: c4 is the same as C4.
  • Octave Range: Standard MIDI range 0 to 9 (default is 4).
  • Omitted Octave: Writing just C will use the current default octave.

3. Modifiers

  • Rest R: e.g., RQ (Wait 1 beat).
  • Dotted .: +50% length. e.g., Q. (1.5 beats).
  • Triplet _T: 2/3 length. e.g., Q_T.
  • Tie +: Sum durations. e.g., H.+E (3.5 beats).

4. Sequential Playback

In WaveCode, melody blocks automatically queue. If you place multiple blocks, they will play one after another instead of overlapping.
