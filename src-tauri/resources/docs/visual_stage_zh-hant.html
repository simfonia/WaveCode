即時分析儀：看見聲音的形狀

WaveCode 的右側面板整合了專業的即時分析儀，讓您在撰寫程式碼的同時，能即時觀察 Rust 引擎產生的原始音訊數據。這不僅是視覺饗宴，更是音訊除錯的重要工具。

1. 示波器 (Oscilloscope)

顯示： 聲音在時間軸上的震盪曲線。

採用「磁滯觸發 (Hysteresis Trigger)」技術，能在演奏高頻音符時依然保持波形穩定、不晃動。您可以直觀地看到正弦波的圓潤或方波的平整。

2. 頻譜分析儀 (FFT)

顯示： 聲音在頻率空間的分佈情況。

透過快速傅立葉變換，將複雜的聲音拆解成不同頻率的強度。您可以觀察到基頻及其諧波，並看見濾波器如何削弱特定的頻率範圍。

視覺化警示與互動

視覺現象 功能說明 意義與對策
波形變紅 (CLIP) 當音訊數值超過 1.0 或低於 -1.0。 代表破音！請降低「音量」積木的百分比，或檢查濾波器的 Q 值是否過高。
ADSR 動畫光點 積木上的黃色跳動光點。 顯示聲音正處於 A-D-S-R 的哪一階段。若不跳動，請檢查樂器 ID 是否正確連結。
多彩頻譜 FFT 長條圖的顏色變化。 顯示低音（左側）到高音（右側）的能量分佈。顏色隨頻率動態生成，提供更佳的空間感。

鍵盤即時演奏功能

在 WaveCode 工作區按下以下鍵盤按鍵，會立即驅動音訊引擎並連動分析儀：

  • [ Q - \ ]：對應 MIDI Note 60 - 81 (鋼琴音階)。
  • 持續按住：光點會停留在 ADSR 的 Sustain 階段。
  • 放開按鍵：聲音進入 Release 階段並緩慢消失。
專家提示： 觀察示波器中的「方波」。由於 Rust 引擎的高保真度，您可以看見完美的水平線；而當您加入「濾波器」後，會看見方波邊緣變圓滑，這正是類比合成器的魅力所在。
