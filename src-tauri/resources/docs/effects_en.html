Audio Effects

In WaveCode, effects are implemented as fundsp nodes that run on a high-priority Rust audio thread. These effects can be placed inside an Instrument Definition to affect its 8-voice polyphonic output, or in the setup block as Global effects.

Signal Chain Logic: The fundsp engine processes nodes in a linear signal chain. The order of blocks determines the final sound texture.

1. Filter (MoogFilter)

A high-quality Rust implementation of the classic Moog low-pass filter. Shapes the brightness and resonance of the sound.

2. Delay & Reverb

Advanced spatial nodes that add depth. Delay creates precise echoes, while Reverb simulates acoustic environments through a complex matrix of reflections.

3. Pitch-Mod & Vibrato

Uses LFO nodes to modulate frequency. Vibrato creates musical rhythmic movement, while Jitter adds high-speed random instability for a Lo-Fi analog feel.

4. Dynamics (Compressor & Limiter)

The WaveCode Engine includes a master limiter by default, but you can add your own dynamics nodes to control individual instrument punch and prevent digital clipping.

Expert Tip: Because WaveCode uses fundsp, you can modulate almost any effect parameter in real-time without introducing audible "zipper noise" or glitches.
