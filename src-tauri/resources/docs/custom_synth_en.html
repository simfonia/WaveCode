Advanced Timbre Design

Go beyond basic waveforms using Additive Synthesis in WaveCode. Our Rust-powered audio engine uses fundsp nodes to build complex harmonic stacks that remain mathematically precise and sonically rich.

Engine Performance: Every custom synth defined in WaveCode automatically benefits from 8-voice polyphony, allowing you to play rich, multi-layered textures without voice stealing.

1. Harmonic Synthesis

Stack frequencies using integer multiples of the fundamental frequency. This is the foundation for simulating natural acoustic timbres such as strings, brass, or woodwinds. By adjusting the amplitude of each harmonic, you can shape the brightness and "nasality" of the sound.

2. Custom Additive Synth

Break the integer rule! WaveCode allows you to define non-integer frequency ratios (e.g., 1.414, 2.718). This is essential for creating inharmonic textures like metallic bells, industrial pads, or complex cinematic sound effects. Each partial is treated as an independent oscillator node within the fundsp signal graph.

3. Optimization Tips

While the Rust engine is extremely efficient, adding hundreds of oscillators to a single instrument can increase CPU load. We recommend using up to 16 partials for a good balance between complexity and performance.
