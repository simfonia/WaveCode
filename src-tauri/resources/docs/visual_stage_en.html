WaveCode Engine & Stage Control

The WaveCode Engine is built on the Rust programming language and the fundsp audio processing library. It provides high-performance, real-time audio synthesis alongside a visual performance stage.

Stage Interaction: The stage is not just for visualization; it acts as a global communication hub between your code and external hardware (MIDI/Serial).

1. Global Programmable Variables

Access and control these system variables in the draw loop or event handlers to create dynamic performances.

VariableTypeDescription
waveScalefloatAmplitude multiplier for the waveform visualization.
fgHuefloatForeground color hue (HSB 0-255).
masterGainfloatThe main output volume control in decibels (dB).
adsrA/D/S/RfloatReal-time envelope parameters affecting all active voices.

2. Keyboard Interaction

Use your PC keyboard as a musical instrument. The following shortcuts are built-in:

  • [ ↑ / ↓ ]: Shift current octave up or down.
  • [ ← / → ]: Cycle through defined instrument containers.
  • [ Backspace ]: Reset octave to default (Octave 4).

3. Technical Foundation

Under the hood, every sound source in WaveCode is a fundsp Node. These nodes are compiled into a signal processing graph that runs on a dedicated high-priority audio thread. This architecture ensures that even during heavy visual rendering, the sound remains clean and jitter-free.
