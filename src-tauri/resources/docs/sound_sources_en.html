Sound Source Creation

This is the laboratory of sound. In WaveCode, all sounds are defined as fundsp nodes within the Rust-based audio engine. Each instrument supports 8-voice polyphony, ensuring smooth performance for complex harmonies.

1. Init fundsp Audio Engine

The master switch for all audio features. Place this block at the very top of your setup block. It initializes the WaveCode Engine and prepares output channels.

2. Define Instrument (Container)

A Hat block used to encapsulate all settings for one instrument. This creates a dedicated signal chain in the fundsp graph.

Note: A container can only have one primary sound source (Waveform, Sampler, or Synth). Lower blocks will overwrite previous ones.

3. Set Waveform

Basic oscillators. Supports Sine, Square, Triangle, and Sawtooth waveforms.

4. Set ADSR

Controls the "energy shape" of each of the 8 polyphonic voices. Must be placed inside an Instrument container.

5. Melodic Sampler

Uses real instrument recordings. The WaveCode Engine automatically selects the nearest sample and performs high-quality pitch shifting in real-time.

Using Custom Samples

WaveCode supports loading your own audio files (.wav, .mp3, .aiff).

Quick Open: Click Settings > Open Sound Samples Folder to manage your assets.
Expert Tip: Parameter changes in WaveCode are processed at the sample level thanks to fundsp, ensuring glitch-free real-time modulation.
