音樂演奏：驅動合成器發聲

「音樂演奏」積木是將程式邏輯轉化為聽覺饗宴的關鍵。WaveCode 採用全自動聲部管理 (Auto-Voice Allocation)，您只需指定音高與時值，引擎便會自動為您分配閒置的聲部進行播放。

1. 演奏音符 (Play Note)

參數：頻率 (Hz)、持續時間 (ms)、樂器 ID

這是最常用的積木。它會啟動指定的樂器 ID，演奏對應的頻率，並在時間結束後自動釋放聲部（進入 Release 階段）。

提示：此積木具備「阻塞 (Blocking)」特性，會等待音符播完才執行下一個積木。

2. 觸發音符 - 不等待 (Trigger Note Async)

參數：頻率 (Hz)、持續時間 (ms)、樂器 ID

與前者不同，它會「立即」啟動聲音並繼續執行後續的積木邏輯。這對於實作和弦 (Chords) 或快速重疊的音效非常有用。

範例：連續放置三個此積木，可以同時發出三個不同頻率的聲音，組成三和弦。

3. 關於頻率與音名

WaveCode 支援兩種指定音高的方式：

  • 原始頻率：直接輸入 Hz 數值（如 440 代表 A4）。
  • 音符轉換器：搭配「音符」積木，可以直觀地選擇 C4、F#5 等音樂符號，系統會自動轉換為精確的頻率數值。
安樂死機制 (Auto-Cleanup)： 當您點擊工具列的「停止」或重新點擊「執行」時，舊的播放任務會立即被中斷，所有運算中的聲部會被強制歸零。這確保了您的實驗環境永遠是乾淨且即時的。

4. 鍵盤即時演奏 (PC Keyboard)

WaveCode 支援使用電腦鍵盤（QWERTY...）進行即時演奏。當您按下鍵盤時，會觸發 EnvelopeManager 的即時動畫。配合示波器，您可以直觀地觀察不同按鍵持續時間對 ADSR 波形的影響。
